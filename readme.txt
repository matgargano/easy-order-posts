Provides a Drag-n-Drop interface to update menu_order for custom post types, posts and pages

== Description ==

Provides a Drag-n-Drop interface to update menu_order for custom post types, posts and pages, two filters are exposed:

easy_order_posts_post_types - modify the post types to apply this plugin to 
easy_order_posts_capability - modify the capability of user to be able to update post_type order
